🧠 Object Detect AI with Low Vision Mode

An AI-powered real-time object detection system designed to assist visually impaired and low-vision users by identifying surroundings and providing accessible feedback.

🚀 Features
🔍 Real-Time Object Detection
Detects multiple objects using AI (YOLO / TensorFlow / OpenCV)
👁️ Low Vision Mode
High contrast visuals
Enlarged bounding boxes and labels
Simplified UI for better visibility
🔊 Voice Assistance
Converts detected objects into speech
Helps users understand surroundings without visual dependency
🎯 Direction Guidance
Provides spatial awareness (e.g., "Chair on your left")
Helps in navigation
📱 User-Friendly Interface
Minimal design for accessibility
Easy toggle between normal and low vision modes
🛠️ Tech Stack
Frontend: React / HTML / CSS / Tailwind
Backend: Node.js / Python (Flask or FastAPI)
AI Model: YOLOv5 / YOLOv8 / TensorFlow Lite
Libraries:
OpenCV
Speech Synthesis API / pyttsx3
WebRTC (for camera access)

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/your-username/object-detect-ai.git
cd object-detect-ai
2️⃣ Install dependencies
For Backend (Python)
pip install -r requirements.txt
For Frontend
npm install
3️⃣ Run the project
Start Backend
python app.py
Start Frontend
npm run dev
🎮 Usage
Open the app in your browser
